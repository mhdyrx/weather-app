# Weather App

A responsive weather application built with Vanilla JavaScript that allows users to search for any city, view current weather information, and save favorite locations using Local Storage.

## Live Demo

🔗 https://mhdyrx.github.io/weather-app/

## Features

- Search weather by city name
- Display current temperature, humidity, and wind speed
- Dynamic weather icons based on weather conditions
- Favorite cities with Local Storage persistence
- Loading spinner while fetching data
- Error handling for invalid city names and failed requests
- Search suggestions for saved favorite cities
- Responsive design

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo API
- Geocode Maps API
- Local Storage
